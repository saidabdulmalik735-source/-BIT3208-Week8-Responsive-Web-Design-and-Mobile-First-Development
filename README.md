# Week 8: Responsive Personal Profile Page

## BIT3208 — Advanced Web Design & Development

### Features
- ✅ Mobile-first responsive design
- ✅ CSS Flexbox layout
- ✅ CSS Grid for skills section
- ✅ Media queries for tablet, desktop, and large screens
- ✅ Responsive profile image (object-fit: cover)
- ✅ Hamburger menu for mobile navigation
- ✅ Print-friendly styles

### Breakpoints
| Breakpoint | Target Devices |
|------------|---------------|
| &lt; 768px | Mobile phones |
| 768px - 1023px | Tablets |
| 1024px+ | Desktops |
| 1440px+ | Large monitors |

### How to Test Responsiveness
1. Open `index.html` in browser
2. Press F12 to open DevTools
3. Click Toggle Device Toolbar (Ctrl+Shift+M)
4. Select different device sizes:
   - iPhone SE (375px)
   - iPad (768px)
   - Desktop (1024px+)
